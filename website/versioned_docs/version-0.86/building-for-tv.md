TV 기기 지원은 기존 React Native 애플리케이션이 JavaScript 코드를 거의 또는 전혀 변경하지 않고 Apple TV와 Android TV에서 작동할 수 있도록 구현되었습니다.

:::warning Deprecated
TV 지원은 [React Native for TV](https://github.com/react-native-tvos/react-native-tvos#readme) 저장소로 이동되었습니다. Apple TV 또는 Android TV 프로젝트에 대한 정보는 해당 저장소의 **README**를 참조하세요.
:::
