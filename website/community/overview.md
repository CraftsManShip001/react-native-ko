React Native는 [수십만 명의 개발자](https://github.com/facebook/react-native/stargazers)로 구성된 방대한 커뮤니티를 기반으로 하며, 매주 [200만 회 이상](https://www.npmjs.com/package/react-native) 다운로드되는 널리 사용되는 기술입니다.

이 섹션에서는 여러분의 필요와 목적에 맞게 React Native 관련 커뮤니티에 참여할 수 있는 방법들을 소개합니다:

- [최신 소식 받기](staying-updated)
- [커뮤니티 참여하기](communities)
- [도움 요청 및 제공하기](support)

React Native 커뮤니티에 참여하는 모든 분들은 [행동 강령](https://github.com/facebook/react-native/blob/main/CODE_OF_CONDUCT.md)의 지침을 따르시기 바랍니다.
