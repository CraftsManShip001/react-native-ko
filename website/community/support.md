React Native 앱에 대한 도움이 필요하다면, 적합한 곳은 필요한 도움의 종류에 따라 다릅니다.

### 저장소

**React Native**의 핵심은 Meta의 React Native 팀이 풀타임으로 작업하고 있습니다. 하지만 커뮤니티에는 핵심적인 기여를 하고 문제를 수정하는 훨씬 더 많은 사람들이 있습니다. 직면한 이슈가 코드 관련 문제라면 [메인 저장소](https://github.com/facebook/react-native/issues)의 공개 이슈를 확인하세요.

기존 이슈를 찾을 수 없다면 [버그 보고 방법](/contributing/how-to-report-a-bug)을 참조하세요.

### 업그레이드 지원

React Native의 한 버전에서 더 최신 버전으로 앱과 라이브러리를 업그레이드할 때 도움이 필요한 경우가 많습니다. 커뮤니티는 몇 가지 중요한 리소스를 함께 만들었습니다:

- [upgrade-helper](https://react-native-community.github.io/upgrade-helper/)는 두 버전 간에 발생하는 모든 변경 사항을 보여주는 도구로, 코드에서 어떤 변경이 필요한지 파악하는 데 도움을 줍니다.
- [upgrade support](https://github.com/react-native-community/upgrade-support)는 앱 업그레이드 시 도움을 요청하고 제공할 수 있는 커뮤니티 주도 저장소입니다.

### Stack Overflow

커뮤니티의 많은 구성원들이 Stack Overflow를 통해 질문을 합니다. **react-native** 태그가 달린 [기존 질문들](https://stackoverflow.com/questions/tagged/react-native?sort=frequent)을 읽어보거나 [직접 질문](https://stackoverflow.com/questions/ask?tags=react-native)하세요!

### Reactiflux Chat

즉각적인 답변이 필요하다면 [Reactiflux Discord](https://discord.gg/JuTwWB8rsy) 커뮤니티를 확인해 보세요. 도움을 주거나 참고할 만한 곳을 안내해 줄 수 있는 React Native 전문가들이 상주하고 있습니다.
