React Native 생태계는 매우 광범위하며, 사람들은 다양한 방식으로 그 일원이 될 수 있습니다. 여기서는 개발자가 참여할 수 있는 다양한 방법의 일부 목록을 소개합니다. 다른 커뮤니티를 알고 있거나 이 페이지를 확장하는 데 도움을 주고 싶다면 [풀 리퀘스트를 제출](https://github.com/facebook/react-native-website/pulls?q=is%3Apr+is%3Aopen+sort%3Aupdated-desc)하세요!

### 지역 커뮤니티

전 세계적으로 React Native 모임이 많이 열리고 있습니다. React 밋업에서도 React Native 관련 내용이 다뤄지는 경우가 많으며, [Meetup](https://www.meetup.com/topics/react-native/)이나 [Eventbrite](https://www.eventbrite.co.uk/d/online/react-native/?page=1) 같은 도구를 사용해 주변의 최근 행사를 찾아보거나 직접 모임을 시작해 보세요!

### 기업 기반 커뮤니티

React Native에 활발히 관여하는 일부 기업들은 자신들이 관리하는 프로젝트를 중심으로 자체 커뮤니케이션 채널을 운영하고 있습니다:

- [Callstack.io의](https://www.callstack.com/) [Discord 서버](https://discordapp.com/invite/zwR2Cdh)
- [Invertase.io의 (예: React Native Firebase)](https://invertase.io/) [Discord 서버](https://discord.gg/C9aK28N)
- [Infinite Red의](https://infinite.red/) [Slack 그룹](https://community.infinite.red/)
- [Expo의](https://expo.dev/) [Discord 서버](https://chat.expo.dev/)

### 독립 커뮤니티

상업적 이해관계를 가진 기업의 후원 없이 자생적으로 형성된 React Native 개발자 커뮤니티입니다.

- [Discord](https://discord.gg/reactiflux)의 [Reactiflux 커뮤니티](https://reactiflux.com)

### 콘텐츠 공유

React Native 태그가 달린 콘텐츠는 다음과 같은 다양한 플랫폼에서 찾아볼 수 있습니다:

- [DevTo 커뮤니티](https://dev.to/t/reactnative)
- [Medium](https://medium.com/tag/react-native)
- [Hashnode](https://hashnode.com/n/react-native)
- [Hacker News](https://hn.algolia.com/?q=react-native)
- [r/reactnative/](https://www.reddit.com/r/reactnative/)

이곳에서는 React Native 프로젝트, 글, 튜토리얼을 공유하고, React Native 관련 주제에 대한 토론을 시작하거나 피드백을 요청할 수 있습니다. (하지만 [메인 문서](https://github.com/facebook/react-native-website)에도 많은 관심을 가져주세요!)
