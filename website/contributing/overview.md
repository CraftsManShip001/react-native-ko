이 사이트는 [React Native 공식 문서](https://reactnative.dev)의 **비공식 한국어 번역본**입니다. 누구나 번역 개선에 참여할 수 있습니다.

## 이런 기여를 환영합니다

- **오탈자·어색한 번역 수정** — 읽다가 이상한 표현을 발견하면 고쳐 주세요.
- **미번역 페이지 번역** — 아직 영어로 남아 있는 문서를 번역해 주세요.
- **오역 신고** — 직접 고치기 어렵다면 이슈로 알려 주세요.

## 문서 위치

번역 문서는 다음 경로에 있습니다.

- 가이드 · 컴포넌트 · API: `website/versioned_docs/version-0.86/`
- 기여 안내(이 문서): `website/contributing/`

수정할 페이지의 `.md` 파일을 찾아 직접 편집하면 됩니다.

## 번역 컨벤션

- **문체**: `~합니다` / `~하세요` 체를 일관되게 사용합니다.
- **코드는 그대로**: 코드 블록, 인라인 코드, `import` 문, 컴포넌트·API 이름(`View`, `Text`, `Animated` 등)은 번역하지 않습니다.
- **고유명사 유지**: React, React Native, iOS, Android 등은 영어로 둡니다.
- **링크 경로 유지**: 링크의 표시 텍스트만 번역하고 경로(URL)는 건드리지 않습니다.
- **헤딩에 `{#앵커}`를 새로 달지 마세요** — MDX 빌드가 깨질 수 있습니다.

## 기여 절차

1. 저장소를 포크하고 새 브랜치를 만듭니다.
2. 문서를 수정하거나 번역합니다.
3. 풀 리퀘스트(PR)를 보냅니다.

이슈와 PR은 [GitHub 저장소](https://github.com/CraftsManShip001/react-native-ko)에서 받습니다.
